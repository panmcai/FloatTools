# 关于本项目

## 项目简介

这是一个浮点数可视化工具，支持多种浮点数格式的可视化展示和转换。

## 支持的格式

### IEEE 754 标准格式
- **FP32 (Single Precision)**: 32位单精度浮点数
- **FP64 (Double Precision)**: 64位双精度浮点数
- **Half (FP16)**: 16位半精度浮点数
- **BFloat16 (E8M7)**: 16位Brain浮点数

### 扩展格式
- **FP8 Training (E4M3)**: 8位训练格式，4位指数，3位尾数
- **FP8 Inference (E5M2)**: 8位推理格式，5位指数，2位尾数
- **FP4 E2M1**: 4位浮点数，2位指数，1位尾数

## 主要功能

1. **位可视化**: 直观展示符号位、指数位和尾数位
2. **实时转换**: 不同格式间的实时数值转换
3. **数学公式**: 展示浮点数的数学表示形式
4. **换算过程**: 详细的浮点数换算步骤说明
5. **特殊值处理**: 支持无穷大、NaN等特殊值
6. **快速运算**: 提供常用的数学运算功能
7. **十六进制支持**: 支持十六进制输入和粘贴

## 技术栈

- **Next.js 16**: React 框架
- **React 19**: UI 库
- **TypeScript**: 类型安全
- **Tailwind CSS 4**: 样式框架
- **shadcn/ui**: UI 组件库

## 特色功能

### CUDA RN 舍入模式
采用与 CUDA 相同的"就近舍入"(Round to Nearest)模式，确保计算结果的一致性。

### 扩展指数范围
- FP8 E4M3 和 FP4 E2M1 扩展了指数范围，支持更大的数值范围
- FP8 E4M3 指数范围: 1-15
- FP4 E2M1 指数范围: 0-3

### 特殊值处理
- 仅当指数全1且尾数为0时判断为 Infinity
- FP8 E4M3 和 FP4 E2M1 不支持 INF/NaN，所有位组合都是有效值

## 使用方式

1. 选择要查看的浮点数格式
2. 在数值输入框中输入十进制数，或在十六进制输入框中输入十六进制值
3. 点击位可以手动修改对应的比特位
4. 使用快速操作按钮快速设置常用值
5. 使用快速运算进行数学运算
6. 切换格式可以查看同一数值在不同格式下的表示
